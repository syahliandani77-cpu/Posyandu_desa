# Posyandu Desa Luk

Website Posyandu Desa Luk adalah platform kesehatan masyarakat yang dirancang untuk mendukung ibu, anak, dan keluarga melalui layanan kesehatan yang mudah diakses.  
Proyek ini dibuat menggunakan **PHP** dan bertujuan untuk menyediakan informasi kesehatan, jadwal layanan, serta laporan kegiatan Posyandu.

## ✨ Fitur Utama
- 📊 **Dashboard Statistik**: Pemantauan balita, ibu hamil, dan kegiatan Posyandu.  
- 👩‍🍼 **Data Balita & Ibu Hamil**: Pencatatan pemeriksaan rutin dan imunisasi.  
- 🍎 **Informasi Gizi**: Edukasi pola makan sehat untuk keluarga.  
- 📅 **Jadwal Posyandu**: Informasi kegiatan dan layanan kesehatan desa.  
- 🔐 **Login & Register**: Sistem autentikasi untuk admin dan pengguna.

## 📂 Struktur Folder
- `index.php` → Halaman utama  
- `dashboard.php` → Tampilan dashboard  
- `balita.php` → Data balita  
- `imunisasi.php` → Data imunisasi  
- `laporan.php` → Laporan kegiatan  
- `login.php` → Halaman login  
- `menu.php` → Navigasi menu  
- `koneksi/` → File koneksi database  
- `layout/` → Template layout (header, footer, dll)

## 🚀 Cara Menjalankan
1. Clone repository ini:
      git clone https://github.com/username/Posyandu_desa.git   
2. Letakkan folder di dalam direktori htdocs (XAMPP) atau www (Laragon).
3. Import database MySQL sesuai file konfigurasi di folder koneksi.
4. Jalankan server Apache & MySQL.
5. Akses di browser:http://localhost/Posyandu_desa
